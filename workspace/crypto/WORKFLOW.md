# Crypto 趋势跟踪工作流 V1

更新时间：2026-03-26

## 目标
用现有能力快速产出一版可执行的加密市场判断，先聚焦：BTC / ETH / DOGE。

使用的数据层：
- polyclaw：看 Polymarket 概率与预期
- okx-dex-market：看价格与市场结构
- okx-dex-signal：看 smart money / whale / KOL 动向
- okx-dex-token：看热点币、风险、流动性
- web-search-pro：补新闻、宏观、监管、ETF 催化

## 一、输出模板（固定）
每次分析都按这 6 行输出：

- 趋势：偏多 / 偏空 / 震荡
- 结构：强 / 一般 / 弱
- 资金：流入 / 分歧 / 流出
- 催化：有 / 弱 / 无
- 动作：观察 / 试仓 / 等回踩 / 等突破 / 减仓
- 风险：最大风险点是什么

## 三、五步工作流

### Step 1：看预期（polyclaw）
目的：看市场在押什么，不先看主观判断。

重点：
- BTC 是否高于 68k / 70k / 72k
- ETH 是否高于 2k / 2.1k / 2.2k
- DOGE 是否存在像样概率盘；没有就标注“缺概率层”
- 月内极端盘：是否到更高价位，是否跌到关键支撑

得到结论：
- 市场在押突破、震荡，还是回撤
- 市场是否相信极端上涨/下跌叙事

### Step 2：看行情结构（okx-dex-market）
目的：确认价格本身是否支持预期。

重点：
- 当前价格
- 24h 涨跌幅
- 是否处于关键整数位附近
- 是强势跟涨，还是弱势反弹

得到结论：
- 图形是强、弱，还是中性震荡

### Step 3：看资金偏好（okx-dex-signal）
目的：看 smart money / whale / KOL 正在买什么。

重点：
- 主流币是否被继续配置
- 资金是否转向高 beta / meme
- 是风险偏好回升，还是保守防守

得到结论：
- 资金偏好是回主流，还是追高风险资产

### Step 4：看标的质量（okx-dex-token）
目的：防止只看涨跌，不看标的本身风险。

重点：
- 热门榜 / 趋势榜
- 流动性
- 市值与成交额
- 持仓集中度
- 风险等级

得到结论：
- 值不值得碰
- 是短打票，还是相对稳一点的方向

### Step 5：看催化（web-search-pro）
目的：补“为什么动”。

重点：
- ETF 资金流
- SEC / 监管
- 美联储、CPI、非农
- 黑客、交易所事故、项目公告

得到结论：
- 有没有逻辑支撑
- 这波能不能持续

## 五、工作原则
- 不在中间位置追单
- 先等确认，再给动作
- 看预期、看结构、看资金、看催化，四项至少占三项才下偏强结论
- DOGE 这类情绪币，不在大盘偏弱时硬抄底
- 没有概率盘的标的，要明确标注“预期层不足”
